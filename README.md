# AI-Market-Analysis-Dashboard-
AI Market Analysis Dashboard adalah aplikasi Web AI untuk membantu trader melakukan analisis teknikal.  Fitur utamanya meliputi: - Dashboard interaktif dengan pilihan pair &amp; timeframe - Chart candlestick (TradingView embed) - Analisis AI Gemini dengan output Bias, Alasan, Area Penting, Skenario, Risiko
